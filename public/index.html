Timestamp API

This microservice allows you to request a date and have it displayed back as a unix timestamp or a plain text, easy to read date.

Example Request

https://timestamp-api-cerebr4l.c9users.io/December%2012,%202016 https://timestamp-api-cerebr4l.c9users.io/1481500800

Example Response

{"unix": 1481500800, "natural": "December 12, 2016"}
